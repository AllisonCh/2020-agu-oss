agu_oss's documentation!
========================

This is an example documentation website for our dummy package ``agu_oss``.

To create your own website, we highly recommend using ``sphinx-quickstart``
and then creating a section of the documentation auto including the API
docstrings from your Python module. We do this with the ``sphinx.ext.autodoc``
extension.

* `Sphinx Quickstart <https://docs.readthedocs.io/en/stable/intro/getting-started-with-sphinx.html#quick-start>`_
* `Sphinx AutoDoc <https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html>`_

We also highly recomend using the Sphinx `Read the Docs Theme <https://sphinx-rtd-theme.readthedocs.io/en/stable/>`_
